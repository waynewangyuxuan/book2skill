Public analyst consensus + notable calls used to judge the arms. (We can't access proprietary JPM/GS/MS PDFs; this is the public proxy — consensus rating, average price target, notable individual calls.) Gathered via web search ~2026-06-16/17.

## Tesla (TSLA)
- **Mixed / no real consensus.** ~61 analysts: ~20 Buy / 17 Hold / 8 Sell (one 27-analyst poll = Hold; a 47-analyst S&P poll = Buy).
- Avg PT ~$398–420.
- **Extreme dispersion** (the defining feature): high $600 (Dan Ives, Wedbush) → low $24.86 (GLJ Research) / $125 (Colin Langan, Wells Fargo). A pure narrative/optionality stock.
- Sources: marketbeat, stockanalysis.com, tipranks, benzinga.

## Disney (DIS)
- **Buy** consensus (42 analysts: 26 Buy / 4 Hold / 1 Sell).

- Sources: marketbeat, benzinga, 24/7 Wall St., public.com.
