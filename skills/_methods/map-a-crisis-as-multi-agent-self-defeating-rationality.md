# Map a crisis as multi-agent self-defeating rationality

## When to use
When forecasting whether a stress will self-correct or cascade, and multiple stakeholder groups react to the same deteriorating signal.

## Method
1. Enumerate the stakeholder groups exposed to the signal.
2. For each, write down its individually-rational defensive response to the signal worsening.
3. Check whether each response feeds back to worsen the signal for the others.
4. If the loop is reinforcing, predict a slow-then-sudden collapse rather than smooth reversion.
5. Note that the same driving input can be value-creating below a threshold and value-destroying above it (phase-dependent).

## Inputs -> Outputs
Inputs: the deteriorating signal; the stakeholder set; each group's rational response. Outputs: a feedback map and a trajectory shape (gradual then abrupt).

## Failure modes / assumptions
- A circuit-breaker (coordinator, backstop) can interrupt the loop.
- Requires correctly enumerating the stakeholders.

## Worked example
Depositors, lenders, and borrowers each rationally pull back as default risk rises, and each move raises the risk further, producing a quiet build then a sudden run.
