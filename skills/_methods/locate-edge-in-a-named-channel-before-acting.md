## When to use
Before committing to any active decision, when you must confirm you actually hold an edge rather than merely an opinion.

## Method
1. Enumerate a small, fixed, exhaustive set of independent channels in which an edge can exist (e.g., a better forecast of the fundamentals, a different valuation framework, or a superior read of crowd sentiment).
2. For each channel, ask whether your view is demonstrably divergent from the reference along that orthogonal dimension.
3. Require a defensible edge on at least one named channel.
4. If no channel genuinely applies, do nothing — absence of locatable edge is a decision to abstain, not a license to act on conviction.

## Inputs → Outputs
- Inputs: your thesis; the reference view in each channel.
- Outputs: a named edge-channel (or an explicit "no edge → no action").

## Failure modes / assumptions
- Asserting a vague holistic edge that cannot be assigned to any channel.
- Double-counting the same insight across channels to manufacture multiple edges.

## Worked example (illustrative, single)
A practitioner likes a name but cannot place the edge in any channel — the forecast matches consensus, the valuation matches the screen, sentiment is neutral — and correctly declines to act.
