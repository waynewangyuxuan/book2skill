# Diagnose a chronic problem by the disabled regulator

## When to use
When a problem that used to self-correct has become newly chronic.

## Method
1. Trace the problem across historical regimes.
2. For each regime, identify the automatic correction (feedback mechanism) that kept it in check.
3. Date the onset of the chronic phase to the institutional break that removed that regulator.
4. Recognize the fix: restoring discipline requires reinstating a binding constraint, not merely offering a new optional channel.

## Inputs -> Outputs
Inputs: the chronic problem; its regime history; the regulator each regime had. Outputs: the removed feedback mechanism and the constraint-based fix.

## Failure modes / assumptions
- The "removed regulator" must actually have been doing the correcting, not just coincident.
- A new regulator may be needed if the old one cannot return.

## Worked example
A persistent imbalance is dated to the abolition of an automatic adjustment mechanism, implying the cure is a new binding constraint rather than a voluntary scheme.
