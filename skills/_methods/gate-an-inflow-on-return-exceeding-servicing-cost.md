# Gate an inflow on return exceeding servicing cost

## When to use
When an inbound flow (foreign capital, borrowing, fresh investment) is presented as unambiguously good for the recipient.

## Method
1. Identify the marginal return the inflow actually funds.
2. Identify its full servicing cost (interest, repayment, repatriation, risk).
3. Treat the inflow as wealth-creating only if return clears cost.
4. Enumerate the narrow qualifying conditions (funds productive investment, bridges a true liquidity gap for a solvent borrower, brings genuinely new technology); absent all of them, classify it wealth-destroying and position for the bust.

## Inputs -> Outputs
Inputs: the inflow; what it funds; its servicing cost. Outputs: wealth-creating vs wealth-destroying verdict, with the qualifying condition (if any).

## Failure modes / assumptions
- Marginal return is hard to observe; revealed deployment is a clue.
- A good inflow can be misallocated downstream.

## Worked example
A surge of cheap external capital that funds low-return projects fails the gate, marking the setup for a later debt crunch.
