## When to use
When a source has reason to flatter or protect one item and you need an honest read on it.

## Method
1. Decline to ask for an absolute evaluation of the item of interest.
2. Ask the source to rank that item within their full comparable set.
3. Read the relative placement — forcing trade-offs against alternatives surfaces the true assessment because praising everything is exposed by the ordering.
4. Cross-check the ranking against other independent rankings of the same set.

## Inputs → Outputs
- Inputs: a biased rater; the full comparable set.
- Outputs: a relative ranking that constrains the flattering bias.

## Failure modes / assumptions
- Accepting absolute praise that costs the rater nothing.
- A set too small to force meaningful trade-offs.

## Worked example (illustrative, single)
Asked to praise one supplier, a buyer says it is excellent; asked to rank all suppliers, the same buyer places it mid-pack, revealing the truth.
