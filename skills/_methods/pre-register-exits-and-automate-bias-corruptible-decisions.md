## When to use
Before committing to a position whose later management will be exposed to the bias that sets in once you have a stake.

## Method
1. Define the falsifiable quantitative milestones the thesis predicts.
2. Set the numeric action / review / stop thresholds and a full exit ladder at entry, before any P&L stake exists.
3. Make the bias-corruptible decisions mechanical so in-the-moment emotion cannot rewrite them.
4. Define both success and failure exit levels so bias cannot rationalize holding a loser.

## Inputs to Outputs
- Inputs: the thesis's predicted milestones; entry conditions.
- Outputs: a pre-registered exit ladder and mechanical stops set before stake.

## Failure modes / assumptions
- Setting exits after the position moves, when bias has already taken hold.
- Overriding mechanical stops with in-the-moment rationalization.

## Worked example (illustrative, single)
Stops set at entry trigger cleanly on an adverse move; without them the practitioner would have rationalized holding the loser.
