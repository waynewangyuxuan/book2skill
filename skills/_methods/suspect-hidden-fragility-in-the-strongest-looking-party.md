# Suspect hidden fragility in the strongest-looking party

## When to use
When one party in a stressed system is taken as the safe haven because its headline position looks strongest.

## Method
1. Identify what the party's strength is contingent on.
2. Check whether that contingency is an imbalance currently unwinding (e.g. a counterparty absorbing its excess).
3. If the strength depends on the imbalance persisting, reclassify it as concealed exposure.
4. Predict it loses later and larger when the imbalance unwinds — not that it is safe.

## Inputs -> Outputs
Inputs: the apparently-strong party; the contingency its strength rests on. Outputs: safe vs concealed-exposure verdict.

## Failure modes / assumptions
- Some strength is genuinely unconditional — don't reflexively invert.
- Requires correctly identifying the contingency.

## Worked example
A surplus economy treated as the safe haven is shown to depend on deficit partners continuing to absorb its excess, so it bears the larger loss when the imbalance reverses.
