# Catch a symmetric law applied in only one direction

## When to use
When an argument invokes a symmetric causal or conservation link to explain one effect but suspends that same link where it would produce an inconvenient consequence.

## Method
1. Identify the link the argument uses (A affects B via a symmetric constraint).
2. Check whether the situation also requires the reverse application (B affects A, or the constraint forcing a consequence elsewhere).
3. If the argument honors the link one way and silently drops it the other, flag selective application as motivated reasoning.
4. Apply the constraint consistently both ways and re-derive the conclusion.

## Inputs -> Outputs
Inputs: an argument using a symmetric link; the reverse case it must also govern. Outputs: whether the link is applied consistently, and the conclusion once symmetry is enforced.

## Failure modes / assumptions
- Some links genuinely are asymmetric; confirm the constraint binds both ways first.
- Distinguish a justified scope limit from convenient suspension.

## Worked example
An argument that a conservation law forces one party to adjust, but exempts the other party the same law constrains, is corrected by enforcing it symmetrically.
