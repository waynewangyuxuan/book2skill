## When to use
Facing a generic, well-solved problem where a mature external methodology already exists.

## Method
1. Identify whether the problem is generic with a mature, proven methodology available.
2. If so, adopt that system wholesale rather than inventing a bespoke one.
3. Value the consistency of applying any sound system over the marginal fit of a custom design.
4. Reserve invention for the problems where you genuinely hold or can build an edge.

## Inputs to Outputs
- Inputs: a generic problem; available mature methodologies.
- Outputs: an adopted external system, with invention saved for edge problems.

## Failure modes / assumptions
- Reinventing a solved process and getting an inconsistent bespoke version.
- Adopting an external system for a problem that genuinely needs a custom edge.

## Worked example (illustrative, single)
A practitioner adopts an established note-taking and review methodology wholesale instead of designing their own, gaining consistency.
