## When to use
Judging whether the analysis behind a conclusion actually exists, when you cannot inspect it directly.

## Method
1. Ask to see the supporting analysis behind a conclusion.
2. Read reluctance to expose it as a signal that the work was likely not done.
3. Recognize that genuine work withstands examination — willingness to be examined proxies for substance.
4. Apply the same test to your own work: if you are reluctant to show your support, you probably have not done enough.

## Inputs to Outputs
- Inputs: a conclusion; the willingness to expose its supporting analysis.
- Outputs: a read on whether the work exists, from transparency-willingness.

## Failure modes / assumptions
- Crediting a conclusion whose support is never shown.
- A legitimate confidentiality constraint mimicking reluctance (account for it).

## Worked example (illustrative, single)
Asked for the model behind a confident call, a source deflects; the practitioner discounts the call accordingly.
