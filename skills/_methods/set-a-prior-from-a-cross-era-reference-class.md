# Set a prior from a cross-era reference class

## When to use
When a focal case is treated as unprecedented or uniquely virtuous, and you need a base rate for its likely outcome.

## Method
1. Strip the brand, era, and nationality off the focal case, reducing it to its structural skeleton.
2. Pool it with structurally identical predecessors across eras and continents.
3. Read the roster's recurring terminal outcome as the prior.
4. Assign the focal case that base-rate outcome and demand strong, specific evidence to override it.

## Inputs -> Outputs
Inputs: the focal case; its structural features; the matching roster. Outputs: a base-rate prior for the focal case's outcome.

## Failure modes / assumptions
- Reference-class membership can be gamed by choosing features; fix the structural skeleton first.
- A genuine structural break can void the precedent — require it to be named.

## Worked example
A fast-growing, subsidized-investment economy is pooled with prior cases sharing that structure, yielding a base-rate terminal outcome of overinvestment then debt crunch.
